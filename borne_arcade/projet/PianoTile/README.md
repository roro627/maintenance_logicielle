# PianoTile - Integration borne

## Lancement
Le menu borne lance `borne_arcade/PianoTile.sh`.

## Runtime
- Python 3
- dependance audio rythmes: `librosa` (installee via `requirements.txt`)
- fallback automatique sans `librosa` pour eviter un crash si la dependance est absente

## Config borne
- Description: `description.txt`
- Commandes: `bouton.txt`
- Persistance score: `highscore`
