# Guide utilisateur TronGame

## Lancement depuis la borne
Le jeu est lance via le menu principal avec le script `borne_arcade/TronGame.sh`.

## Commandes borne
Les commandes officielles sont decrites dans `bouton.txt`.

## Retour menu
A la fin de la partie, le processus se ferme et le menu principal reprend la main.

## Depannage rapide
- Si le jeu ne se lance pas, verifier `python3` et `pygame`.
- Executer `./scripts/tests/lancer_suite.sh` depuis la racine.
