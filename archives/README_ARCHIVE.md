# Archives du projet

## Objectif

Ce dossier centralise les anciennes versions et fichiers legacy
qui ne doivent plus rester dans les repertoires actifs.

## Regles

1. Conserver l arborescence d origine dans `archives/versions_legacy/`.
2. Ajouter une note de contexte pour tout nouvel element archive.
3. Ne jamais archiver de fichier sous `MG2D/` par deplacement manuel:
   MG2D suit sa propre regle d integrite et se resynchronise via upstream.
